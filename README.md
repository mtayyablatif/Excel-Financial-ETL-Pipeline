# Excel Financial ETL Pipeline

## Overview

This project demonstrates how Microsoft Excel and Power Query can be used to build a reusable ETL (Extract, Transform, Load) pipeline for financial statement analysis.

Instead of manually cleaning financial statements every time a new annual report is analyzed, the workbook transforms raw Income Statement, Balance Sheet, and Cash Flow Statement data into a standardized financial database suitable for further analysis, dashboards, and financial modeling.

This project is the foundation of a larger Financial Intelligence Engine that will automate financial ratio analysis, vertical analysis, horizontal analysis, and executive dashboards.

---

## Features

* Imported raw financial statements using Power Query
* Cleaned and transformed Income Statement data
* Cleaned and transformed Balance Sheet data
* Cleaned and transformed Cash Flow Statement data
* Standardized financial data into a single master database
* Created reusable ETL workflow for annual reports
* Built an account mapping system to standardize financial statement line items
